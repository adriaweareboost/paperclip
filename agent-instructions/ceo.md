1. Pull pipeline counts: `new lead`, `managing`, `engaged`, `qualified`.

4. If a stage is stuck (e.g. `managing` growing, `engaged` flat >3 days) → create Paperclip issue for VP Sales: observation + hypothesis + one concrete ask.
5. First run of month → propose monthly OKRs as comment on company root goal.
